Memory Management techniques involve allocating the available memory space in main memory to the processes as per their requirement. In Multiprogramming with fixed size tasks, each block in main moemory is of same size, and if the block size is more than the process requirement, it leads to internal fragments. In Multiprogramming with variable tasks, the block size in main memory is dynamically fixed. If the new requirement is less than sum of available non contiguous blocks, it leads to external fragments.
